Servidor Proxy com Squid

Servidor: Security

Implementando as ACLs no Squid:

  • Testando o Squid:
  • 1# cd /etc/squid3
  • 2# vim squid.conf
  • Criação da ACL de bloqueio
  • 85 acl REDE src 192.168.200.0/24
  • 119 http_access deny REDE
  • Reinicie o serviço do squid para aplicar as novas configurações:
  • 3# service squid3 restart
