Laboratório Dexter

  • Configure o fstab garantindo a montagem na inicialização do sistema:
  • 1# blkid
  • 2# blkid | grep publico | awk '{ print $2}' | sed -­e s/\"//g >> /etc/fstab
  • 3# vim /etc/fstab
  • UUID=84f793e5-fa82-42e0-b4f2-058a87a2e317 /srv/samba/publico ext4 defaults 0 0
  • 4# umount /srv/samba/publico ; mount -a ; mount

<Atenção no uso do operador “>>” para não apagar o fstab!>
