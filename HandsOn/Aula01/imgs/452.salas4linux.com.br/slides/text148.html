Instalando o Webmail RoundCube:

O Roundcube é um cliente IMAP no formato webmail, com uma interface bem organizada e suporte a vários recursos, como manipulação de pastas, busca de mensagens e verificação ortográfica.

1 – Faça a instalação do RoundCube:

1# yum install roundcubemail.noarch

2# cd /etc/httpd/sites

3# cp intranet.conf webmail.conf
