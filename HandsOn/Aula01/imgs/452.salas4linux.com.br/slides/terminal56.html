Servidor DHCP

DHCP Estático:

  • 1# vim /etc/dhcp/dhcpd.conf
  • host linux-interna {
  • hardware ethernet 00:00:00:00:00:00;
  • fixed-address 192.168.200.10;
  • }
  • 2# /etc/init.d/isc-dhcp-server restart
  • Faça uma nova solicitação na Linux Interna:
  • 3# dhclient -r ; dhclient -v eth0
