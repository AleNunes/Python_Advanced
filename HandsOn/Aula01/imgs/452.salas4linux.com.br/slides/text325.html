Servidor FTP

Criando usuário do FTP:

  • Crie o usuário webmaster:
  • 1# adduser webmaster -d /var/www/html -g apache
  • Defina a senha do usuário: 2# passwd webmaster
  • Defina o dono e grupo do diretório como o grupo apache: 3# chown -R apache:apache /var/www/html/
  • Adicione permissão no diretório:
  • 4# chmod -R 775 /var/www/html/

Servidor: WebServer Interno


Notes:

O comando adduser pode ser utilizado com diversos parâmetros para a criação de usuários. Foram utilizados as opções:

# adduser webmaster -d /var/www/html -g apache -s /bin/false

-d Para definir um diretório pessoal (home) para o usuário

-g Para definir o grupo principal que o usuário ira fazer parte.

-s Para definir o shell do usuário.
