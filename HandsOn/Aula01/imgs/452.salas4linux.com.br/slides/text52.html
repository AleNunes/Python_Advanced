Servidor DHCP

Servidor: Linux Interna

Cliente DHCP:

A máquina Linux Interna está com o IP estático, portanto precisamos alterar a configuração no arquivo de redes, para que a interface eth0 não tenha mais um ip estático e sim um ip atribuído pelo servidor DHCP.

1# su - root

2# vim /etc/network/interfaces

auto eth0

iface eth0 inet dhcp

post-up route add -net 192.168.200.128/25 gw 192.168.200.129
