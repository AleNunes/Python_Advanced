Configurando a Base de Dados do Roundcube:

1 – O arquivo INSTALL da documentação do RoundCube indica como configurar a base de dados em todos os SGBD`s suportados por ele:

1# vim +87 /usr/share/doc/roundcubemail-1.1.5/INSTALL

2# mysql -u root -p

CREATE DATABASE roundcubemail /*!40101 CHARACTER SET utf8 COLLATE utf8_general_ci */;

GRANT ALL PRIVILEGES ON roundcubemail.* TO roundcube@localhost IDENTIFIED BY '4linux';

FLUSH PRIVILEGES;
