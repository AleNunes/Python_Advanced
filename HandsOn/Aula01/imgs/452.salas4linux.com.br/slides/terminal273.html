Liberando Squid no Firewall:

  • Libere a porta 3128 do Squid e 389 do LDAP no arquivo de configuração do Firewall:
  • 1# vim +109 /etc/rules
    • 110 iptables -t filter -A INPUT -p tcp -s $LAN -d $FWLAN --dport 3128 -j ACCEPT
      • ...
    • 118 iptables -t filter -A INPUT -p tcp -s $DATACENTER --sport 389 -d $FWLAN -j ACCEPT
  • 2# service firewall restart
