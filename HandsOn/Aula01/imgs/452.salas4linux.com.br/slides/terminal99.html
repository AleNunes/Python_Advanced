Notes:

ServerRoot "/etc/httpd": Configuração do Apache

<IfModule prefork.c>: Se for prefork usa essa configuração

StartServers 8 : Quantos processos instancia , para ficar esperando se alguém for acessar o site.

MinSpareServers 5 : Se ninguém acessar o site, quantos processos você quer manter no servidor. (Processos ocioso)

MaxSpareServers 20: Suponha que veio 50 pessoas no site, vai atender as 50 e dps o mínimo vai cai para 20 processos.

ServerLimit 256: Quantos processos ele vai instanciar até 256.

MaxClients 256: No máximo eu vou ter 256 clientes simultâneos.

MaxRequestsPerChild 4000: O máximo que cada processo vai pode fazer de request (4000 cliques, ou chamados próprias do site).

Listen 80: Porta que o Apache irá ouvir.

User apache: Usuário do Apache

Group apache: po do Apache

ServerAdmin root@admin: E-mail do administrador do Apache

ServerName 127.0.0.1:80: Nome do host e a porta.

DocumentRoot "/var/www/html": Diretório dos sites.
