• Dentro do arquivo smb.conf é possível criar restrições no samba de forma que, sejam aceitas requisições de acesso apenas a partir de uma interface:
  • Abra o arquivo smb.conf e descomente as linhas abaixo:
  • 1# vim /etc/samba/smb.conf
  • 8 interfaces = eth0
  • 9 bind interfaces only = Yes
  • 2# systemctl restart sernet-samba-ad
  • 3# samba-tool testparm
