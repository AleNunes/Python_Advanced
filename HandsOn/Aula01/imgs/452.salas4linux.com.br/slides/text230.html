Servidor Samba

Introdução:

  • O Samba nasceu de uma necessidade de seu criador Andrew Tridgell que precisava montar um espaço em disco para um servidor Unix em seu PC, rodando o sistema operacional MS-DOS;
  • Para isso, Tridgell escreveu um sniffer (programa para captura de tráfego de dados) que permitisse analisar o tráfego de dados gerado pelo protocolo Netbios, utilizado pelo Windows em compartilhamentos de dados.

Andrew Tridgell em 2005.
