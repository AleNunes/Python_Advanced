Servidor FTP

Cliente para o servidor FTP:

  • Para facilitar a operação de arquivos, como enviar e receber arquivos com o suporte a interface gráfica, existem programas que funcionam como clientes FTP, que fazem conexão com servidor FTP para auxiliar na gerência de arquivos para um FTP.
  • Como exemplo de cliente FTP: FileZilla (http://filezilla-project.org)
