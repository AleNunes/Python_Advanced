Servidor FTP

Enviando arquivos para o servidor FTP:

  • Vamos enviar o conteúdo para o ftp. Criando um diretório para o site. E enviando vários arquivos:
  • ftp> mkdir site
  • ftp> cd site
  • ftp> !ls
  • ftp> mput index.html pagina*.html
  • ftp> !ls

Servidor: Linux Interna

Notes:

Os comandos Executados com ! Na frente do comando exiber conteúdo da máquina cliente.

Exemplo.

ftp> ls - exibe o conteúdo do FTP.

ftp > !ls - exibe o conteúdo da maquina cliente que se conectou ao FTP.
