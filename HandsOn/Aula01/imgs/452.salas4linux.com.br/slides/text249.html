Gerenciar usuários no Samba 4:

A sintaxe de uso do comando samba-tool para gerenciar usuários e grupos:

    • samba-tool user list
    • samba-tool user add nome.sobrenome
    • samba-tool group list
    • samba-tool group add grupo
    • samba-tool group addmembers grupo nome.sobrenome
