Servidor Samba

Introdução:

  • O Samba é um serviço Linux que permite o gerenciamento e compartilhamento de recursos em redes formadas por computadores com o Windows;
  • Assim, é possível utilizar um sistema Linux como servidor de arquivos, servidor de impressão, entre outros. Este serviço irá se “apresentar” para a rede de forma que o Sistemas Operacionais Windows o interprete como um Servidor Windows.
