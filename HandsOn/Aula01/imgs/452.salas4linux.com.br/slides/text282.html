Servidor Proxy com Squid

Servidor: Security

Configurando o Squid Analyzer:

  • Faça a compilação do Squid Analyzer:
  • 1# apt-get install make perl
  • 2# cd /root/squid
  • 3# tar -xf squidanalyzer-6.5.tar.gz -C /usr/src
  • 4# cd /usr/src/squidanalyzer-6.5
  • 5# perl Makefile.PL
  • 6# make && make install
