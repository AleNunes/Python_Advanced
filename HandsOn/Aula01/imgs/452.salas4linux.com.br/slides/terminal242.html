Servidor Samba

Conhecendo o Arquivo smb.conf:

  • Uma seção sempre é iniciada por seu nome dentro de colchetes [ ] e continua até que um novo conjunto de colchetes demarque a próxima seção;
  • Os parâmetros das seções possuem o seguinte formato: name = value;
  • o arquivo é baseado em linhas individuais, cada nova linha representa um novo parâmetro; estes valores não são CASE SENSITIVE.
