Servidor Samba

Smbclient:

O comando smbclient é um cliente utilizado para acessar compartilhamentos via protocolo SMB.

Sua principal função no samba é testar o funcionamento de um compartilhamento via terminal.

1 – Faça um teste de requisição utilizando comando smbclient:

1# smbclient -L fileserver -U%

Servidor: FileServer
