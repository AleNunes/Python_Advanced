Gerenciar usuários no Samba 4:

CENÁRIO 3: O Samba 4 utiliza seu próprio sistema de permissões baseados nos SIDS de sua base de usuários, com valores diferentes dos UIDS dos colaboradores da empresa Dexter Courier.

SOLUÇÃO: Vamos criar um script para substituir em sua base de mapeamento, o valor do atributo "xidNumber" pelo valor do UID de cada usuário cadastrado na base UNIX.
