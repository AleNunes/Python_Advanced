Pergunta LPI

Quais dos seguintes daemons são necessários para executar um Servidor NFS?

A. Portmap

B. nfsiod

C. mountd

D. xinetd

Após configurar um sistema de arquivos em /etc/exports, qual comando deve ser executado para que a nova configuração seja aplicada?

Resposta: exportfs -a


Notes:

Alternativas Corretas: A & C

Dentro das opções listadas na pergunta LPI, apenas duas delas representam daemons do NFS, são elas:

Portmapper → Daemon responsável por converter processos RPC em números no protocolo TCP/IP.

Status → Daemon que contem informações sobre o compartilhamento a fim de evitar erros;

______________________________________________________________________

REPOSTA CORRETA: exportfs -a

O comando exportfs é utilizado para gerenciar a tabela de exportações do NFS, utiliza-se a opção -a para exportar os compartilhamentos NFS.
