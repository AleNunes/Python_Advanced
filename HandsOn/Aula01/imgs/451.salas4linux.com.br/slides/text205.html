Shell Script

Comando test:

O comando test é usado para realizar testes de condicionais (strings, matemáticas e em arquivos) em “Shell Script”.

Exemplo com strings:

1# test “curso” = “curso” ; echo $?

Exemplo com variável:

2# test -z $CURSO ; echo $?

Exemplo com expressões matemáticas:

3# test 7 -eq 7 ; echo $?

Servidor: WebServerInterno


Notes:

Acima mostramos algumas formas de se testar as condicionais utilizadas dentro da estrutura “se”. Lembre-se que podemos usar as condicionais tanto dentro, quanto fora da estrutura “se”, depende do caso e do meio.
