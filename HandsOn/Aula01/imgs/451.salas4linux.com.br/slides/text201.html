Shell Script

Expressões Matemáticas com Shell Script:

Vamos fazer um script que calcula a média de um aluno:

Servidor: WebServerInterno

1# vim script1.sh

#!/bin/bash

echo “Digite seu nome:”

read ALUNO

echo “Por favor, digite sua primeira nota:”

read N1

echo “Agora digite sua segunda nota:”

read N2

SOMA=$(expr $N1 + $N2)

MEDIA=$(expr $SOMA / 2)

echo “Sua média $ALUNO é: $MEDIA”


Notes:

Expr

Muitas vezes quando fazemos “scripts”, precisamos de uma função que faça o trabalho das operações matemáticas básicas como soma, divisão, multiplicação e subtração. Em “shell script” podemos usar o comando “expr” para realizá-las.
