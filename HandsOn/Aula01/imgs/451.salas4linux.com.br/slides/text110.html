Implementando Controle de Acesso ao Sistema com sudo

1 – Instale o pacote sudo no servidor Fileserver:

1# yum install sudo

2 – Abra o arquivo de configuração do comando sudo:

2# export EDITOR=vim

3# visudo ou vim /etc/sudoers

O comando visudo permite editar o arquivo /etc/sudoers.Sua vantagem em relação a edição direta através de um editor de textos está no fato do comando visudo não permitir que mais de um usuário edite o arquivo ao mesmo tempo, além de indicar erros de sintaxe na edição. Para alterar o editor padrão do comando visudo basta alterar o valor da variável EDITOR
