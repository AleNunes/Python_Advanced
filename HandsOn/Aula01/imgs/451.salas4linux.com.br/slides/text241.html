Gerenciando de dados SQL

  • É extremamente importante ter um dump da base de dados, para realizar a tarefa segue, abaixo, a instrução:
  • Na linha de comando execute:

Onde:

-h localhost: Host a ser feito o dump

-u root: Usuário que irá realizar o dump

-p 4linux: Senha para se conectar ao banco

Backup: Nome do banco de dados a ser feito o dump

1# mysqldump -h localhost -u root -p4linux backup > backup.sql

2# cat backup.sql

  • Para restaurar um backup execute:

1> mysql -h localhost -u root -p4linux backup < backup.sql
