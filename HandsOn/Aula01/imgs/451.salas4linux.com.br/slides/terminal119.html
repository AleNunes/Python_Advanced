Notes:

IPV4

No IPV4, os endereço IP são compostos por 4 blocos de 8 bits (32 bits no total), que são representados através de números de 0 a 255, como "200.156.23.43"ou "192.168.1.1".

Endereços Reservados

As faixas de endereços começadas com "10", com "192.168" ou com o range que vai de "172.16" até "172.31" são reservadas para uso em redes locais e por isso não são usados na internet.

Classes D e E:

Embora não muito comentadas a existência das classes D e E, elas possuem uma característica um pouco diferente das demais, sendo a classe D um endereço de multicast, onde é possível criar um grupo de Ips que terão um mesmo alias, seu range de Ips varia de 224.0.0.0 até 239.255.255.255. Já a classe E foi destinada para um uso futuro, possuindo o range de Ips de 240.0.0.0 até 255.255.255.255.
