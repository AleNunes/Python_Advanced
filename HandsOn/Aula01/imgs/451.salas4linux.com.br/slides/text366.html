Recuperação de Senhas de Root CentOS 7:

4 - Pressione Ctrl + x para inicializar o sistema com os parâmetros alterados.

5 - O sistema de arquivos está montado read-only no /sysroot, você não terá permissão para alterar a senha, portanto remonte o diretório com permissão de escrita:

# mount -o remount,rw /sysroot

6 - Alterar raiz do sistema de arquivos da seguinte forma:

# chroot /sysroot

7 - Altere a senha do root

# passwd

8 - Reinicie a máquina
