Notes:

Diferencial

O “backup” diferencial visa salvar apenas as diferenças em relação ao ultimo “backup” completo (Full), por exemplo: Um “backup” full acontece no domingo. O diferencial salvará os dados de domingo para segunda, de domingo para terça, de domingo para quarta, de domingo para quinta, de domingo para sexta e de domingo para sábado, ou seja, até chegar no próximo “backup” full.

Incremental

Diferente do diferencial, o incremental, faz apenas os incrementos, assim gerando um volume menor de dados. Se o “backup” completo foi gerado no domingo, ele salva de domingo para segunda, de segunda para terça, de terça para quarta e assim até o próximo “backup” full chegar.
