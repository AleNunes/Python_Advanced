Pergunta LPI

Qual comando inicia o controle e monitoração das cotas de disco após estarem corretamente configuradas? (especifique somente o comando)

Resposta: quotaon

Qual é o comando usado para editar novas cotas de disco para um determinado usuário ou grupo do sistema? (especifique somente o comando)

Resposta: edquota

Notes:

RESPOTA CORRETA: quotaon

Após configurarmos um sistema de quotas o comando quotaon deve ser utilizado para ativar o controle de quotas no disco, assim como o comando quotaoff permite desativar um sistema de quotas sem alterar sua configuração ou sem qualquer outro tipo de intervenção no disco.

____________________________________________________________________

RESPOTA CORRETA: edquota

O comando edquota permite a edição de um sistema de quotas de usuários ou grupos.
