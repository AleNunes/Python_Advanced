Shell Script

Estrutura do Shell Script:

#!/bin/bash

#Primeiro Shell

echo “Hello World!”

echo

clear

Servidor: WebServerInterno

Utilizando o interpretador de comandos bash.

Comentário (Não será executado)

Exibe na tela a frase “Hello World!”

Echo vazio pula uma linha

Limpa a tela

NOTA: Note que todos os comandos podem perfeitamente serem executados em sua linha de comando.

2# file shell.sh

3# chmod +x shell.sh

4# ./shell.sh

1# vim shell.sh

Notes:

Entendendo o script

O modelo acima apresenta um “script” bem simples que executa 3 comandos: Limpar a tela (cd ); limpar a tela (clear); chamar o programa calendário (cal); imprimir a data (date) .

A primeira linha, na qual aparece um comentário, possui uma característica um tanto estranha. Na verdade, a primeira linha de um “script”, indica qual será o interpretador daquele “script”. Em nosso exemplo será o programa “/bin/bash”, uma “shell”.

Se estivéssemos criando um “script” com a linguagem de programação “Perl”, a primeira linha seria algo como /usr/bin/perl.
