Compactadores, Empacotadores e Procedimento de Backup

Empacotador Cpio:

3 – Restaurar o backup do /etc:

1# mv /etc/ /tmp

2# cpio -iv < /backup/bkp_etc.cpio

4 – Adicione um usuário no sistema:

3# adduser <seunome>

4# tail -1 /etc/passwd

Opções do CPIO:

-i → Extrair backup;

-o → Criar backup;

-t → Mostrar uma tabela com o conteúdo do backup;

-F → Essa opção especifica o arquivo de backup;

-d → Cria diretórios se necessário;

-v → Mostrar o que está fazendo em detalhes;

-u → Sobrescreve arquivos existentes.

Servidor: WebServerInterno

Notes:

Funções de Backup

Compactar e empacotar arquivos, são funções extremamente necessárias quando falamos de “backup”. Podemos ter diferentes tipos de “backup”, são eles:

Completo (Full)

Visa salvar todos os dados, mesmo o que já foram salvos anteriormente, assim fazendo um “backup” completo de todos os objetos em questão.
