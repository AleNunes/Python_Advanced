A Variável $n

Em algumas situações, pode ser necessário fornecer parâmetros para um “script”. Por exemplo, para criar um script que recebe um determinado valor, sendo que este valor não é fixo, e pode variar conforme a data ou o local de execução do script.

Assim para que você não tenha de mexer no script a cada execução, o valor a ser recebido deverá ser passado no momento da execução. Como fazer isso?

Uma das alternativas para receber este valor é através de variáveis especiais conforme o modelo acima.
