Shell Script

Alguns comandos utilizados no Shell:

Comando

Descrição

Sintaxe

echo

Exibe o texto na tela

echo “texto a ser mostrado”

sleep

Dá um tempo antes de executar o próximo comando

sleep segundos exemplo: Sleep 1

read

Recebe o valor de uma variável

read variável exemplo:

read dados

>

Escreve em um arquivo-texto

(apagando o que estava lá)

echo “texto” > /home/luiz/arquivo

>>

Escreve em um arquivo-texto

(última linha, não apaga)

echo “texto” >> /home/luiz/arquivo

&

Roda o comando em 2º plano

Comando&

exit

Sai do script

exit

touch

Cria um arquivo vazio

touch nome_do_arquivo
