Data e Hora do Sistema e Servidor de NTP

Implementando um Servidor de NTP:

3 – Dentro do arquivo de configuração, altere o conteúdo das linhas 21 a 25 deixando no seguinte padrão:

21 server 0.debian.pool.ntp.org iburst

22 server 1.debian.pool.ntp.org iburst

23 server a.ntp.br iburst prefer

24 server b.ntp.br

25 server c.ntp.br

Parâmetros adicionais:

→Iburst: Esta opção aumenta para 8 o número de pacotes enviados durante a sincronização inicial de data e hora.

→ prefer: A opção prefer força a verificação da resposta dada pelo servidor preferido, se essa resposta diferir muito da resposta dos demais servidores ela será descartada.
