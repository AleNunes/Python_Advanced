Compilação de Kernel

  • Um passo extremamente importante antes de configurar o nosso “kernel” é sempre adicionar uma EXTRAVERSION afim de organizar uma eventual estrutura de módulos no “/lib”:
  • 1# vim Makefile
  • EXTRAVERSION = -v1

Máquina Linux Externa

DICA: Em distribuições Debian o conjunto de pacotes abaixo pode ser substituido pelo pacote: build-essential, no exemplo abaixo, bastaria instalar este pacote e o pacote libncurses5

  • No Debian instale os pacotes necessários para a compilação:
  • 2# aptitude install make gcc g++ autoconf libncurses5 libncurses5-dev ncurses-base ncurses-bin ncurses-term
