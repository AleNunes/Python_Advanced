Visualização de Logs do Sistema

Entendendo as Mensagens de Log:

A norma NBR ISO/IEC 27002 recomenda no item 10.10.1 as seguintes características, para um sistema de logs:

  • Identificação dos usuários;
  • Datas e horários de entrada e saída de terminais;
  • Hostname ou endereço IP, para serviços acessados via rede;
  • Registro das tentativas de acessos aceitos e rejeitados.
  • O padrão FHS define que um sistema linux armazenará os seus logs no diretório /var/log

Notes:

Arquivos de log não são arquivos de erros!

Arquivos de logs são utilizados essencialmente para análise e resolução de problemas, ou seja, verificação de falhas. Além disso provêm outras finalidades tão importantes quanto essas, como auditoria de servidores e extração de relatórios.
