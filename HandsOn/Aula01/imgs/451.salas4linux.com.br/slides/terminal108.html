Implementando Controle de Acesso ao Sistema com sudo

O comando sudo:

Servidor: Fileserver

  • Além da restrição ao uso do comando su através do PAM, uma medida de segurança muito interessante é a implementação do programa sudo "Substitute User Do";
  • O programa sudo permite delegar funções administrativas a qualquer usuário sem, que este mesmo usuário precise obter a senha de administrador;
  • Distribuições GNU/Linux como Ubuntu ou Fedora adotam o uso do sudo, delegando esse poder ao usuário criado no momento da instalação do sistemas.
