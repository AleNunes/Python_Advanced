Pergunta LPI

O diretório /etc/skel é utilizado para:

A. Pelo comando passwd

B. Pelo comando pwconv

C. Pelo comando useradd

D. Pelo comando pwunconv

Resposta: Alternativa C

Notes:

Alternativa C: REPOSTA CORRETA!

Ao criar um usuário através do comando useradd ele fará uma cópia do conjunto de pastas e arquivos definidos no diretório /etc/skel para gerando todo o escopo do diretório home do usuário recém criado.
