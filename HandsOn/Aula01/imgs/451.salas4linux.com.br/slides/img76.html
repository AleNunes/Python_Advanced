Setquota

O comando setquota pode ser muito útil, principalmente por não exigir interatividade, ou seja, pode ser executado facilmente dentro de um script.

Na execução acima foram utilizados os seguintes parâmetros:

-u → Indica que será definida quota para um usuário

annie.dee → Usuário que receberá os valores da quota

50000 → Soft para espaço disponível para uso

60000 → Hard para espaço disponível para uso

100 → Soft para número de arquivos

110 → Hard para número de arquivos

/srv → Partição para aplicar a quota
