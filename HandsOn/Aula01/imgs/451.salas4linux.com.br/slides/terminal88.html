Implementando Controle de Acesso ao Sistema com PAM

Introdução Teórica – Gerenciadores do PAM:

  • O tipo define o modelo de autenticação utilizada pelo módulo, módulos de mesmo tipo podem ser interligados criando um conjunto de critérios para que determinada autenticação ocorra.
  • Abaixo os tipos de autenticação do PAM:
  • → account
  • → auth (authentication)
  • → password
  • → session

Notes:

Descrição dos Tipos de Gerenciadores

→ account: Verifica se um usuário possui permissão para acessar um serviço ou recursos, se a senha não expirou, etc;

→ auth (authentication): Determina a autenticidade do usuário, autenticão via senha, biometria ou qualquer outro método;

→ password: Utilizado para ações específicas referentes a alterações de identificação (Geralmente por senha);

→ session: Procedimentos a serem realizados ao iniciar ou finalizar sessão ou seja, depois da autenticação do usuário.
