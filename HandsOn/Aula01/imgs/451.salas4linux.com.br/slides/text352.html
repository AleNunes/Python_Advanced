Qual é o formato correto de numeração de versão do kernel Linux?

A. 9.04

B. 2009

C. 2.6.31

D. 2008 Server

Notes:

Alternativa C: RESPOSTA CORRETA!

A numeração de Kernel segue o formato Verison (Versão principal) + Patch (Número de revisão principal) + Subversion (Número de revisão secundário) logo o formato correto seria: 2.6.31 conforme descrito na alternativa C.
