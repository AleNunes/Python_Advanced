Data e Hora do Sistema e Servidor de NTP

Implementando um Servidor de NTP:

Além ajustar a hora com base em um servidor, é possível criar uma estrutura própria com servidor NTP. Configuraremos a Audit para trabalhar como servidor NTP interno da empresa Dexter.

1 – Instale o servidor ntp provido pelo pacote ntp:

1# apt-get install ntp

2 – Abra o arquivo /etc/ntp.conf para configurar o servidor:

2# vim /etc/ntp.conf

Servidor: Audit

Notes:

Com qual nível devo sincronizar?

A menos que estejamos montando um servidor para ser um "stratum 1", 2 ou 3, nunca devemos utilizar os servidores "stratum 1"ou 2 para sincronizarmos nossos servidores; mas sim acessar um "stratum 3". Dessa forma deixamos os níveis mais baixos para as máquinas que realmente precisam acessá-los.
