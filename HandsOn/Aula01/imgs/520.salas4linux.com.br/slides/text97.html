Entendendo o ORM

<número>

ORM significa Object Relationship Mapping. Serve para abstrair a interação entre a aplicação e o banco de dados, poupando o desenvolvedor de ter que executar comandos em SQL.

Utilizando um ORM podemos criar Classes que serão presentadas como Tabelas no Banco de Dados, e os Atributos dessas Classes serão as colunas da Tabela no Banco de Dados.

pip install sqlalchemy
