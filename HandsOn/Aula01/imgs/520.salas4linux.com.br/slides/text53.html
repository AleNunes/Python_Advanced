Módulo Paramiko

<número>

Existe um módulo em Python que serve especificamente para fazer conexões via SSH, que é muito usado no mundo DevOps.

Para instalar, execute o seguinte comando:

pip install paramiko

Este módulo é usado por ferramentas como Ansible e Fabric que fazem conexões fica ssh para o gerenciamento de configurações.
