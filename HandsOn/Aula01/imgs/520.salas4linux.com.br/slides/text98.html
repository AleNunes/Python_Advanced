Conhecendo o SQLAlchemy

<número>

O SQLAlchemy é um kit de ferramentas para banco de dados e mapeamento do objeto relacional, que disponibiliza aos desenvolvedores muitos recursos e flexibilidade ao trabalhar com SQL.

Ele é usado em grandes projetos, como o OpenStack, cuja maioria dos componentes é escrita em python e faz a interação com bancos de dados relacionais, onde é usado o SQLAlchemy para fazer a interação com qualquer banco de dados.

Link: http://www.sqlalchemy.org/organizations.html#openstack
