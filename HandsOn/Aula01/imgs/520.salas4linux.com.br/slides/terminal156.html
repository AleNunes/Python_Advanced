Herança Múltipla

<número>

  • A herança múltipla no Python é declarada da seguinte maneira:

class Fisico(Servidor,Acesso):

def __init__(self):

self.total_slots = 4

self.slots_ocupados = 1

#serão adicionados somente esses dois novos atributos

  • Agora a classe físico já herda as características das duas classes.
  • Também podemos usar o conceito de polimorfismo para diferenciar os acessos.
