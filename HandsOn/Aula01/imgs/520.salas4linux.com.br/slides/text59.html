Módulo Datetime

<número>

Para fazer o cálculo de datas, utilizamos o timedelta, que recebe como parâmetro a quantidade de dias que deve ser calculada.

from datetime Import datetime, timedelta

print datetime.now() + timedelta(7)

Nesse exemplo o resultado será a data de hoje mais sete dias. Isso é muito importante quando queremos automatizar a remoção de um serviço.
