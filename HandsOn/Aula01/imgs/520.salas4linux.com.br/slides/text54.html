Para usar o paramiko precisamos instanciar um objeto do tipo SSHClient.

from paramiko.client import SSHClient

Import paramiko

client = SSHClient()

Esse objeto tem todos os métodos e propriedades necessárias para que possamos executar as conexões via ssh.
