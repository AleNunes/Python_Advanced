Utilizando a instrução break faz-se com que o a execução saia imediatamente do loop e vá para a próxima instrução fora do laço.

servidores = ["dns","database","webserver","ldap","mongdb"]

for s in servidores:

if s == "ldap":

print "O servidor ldap nao pode estar na lista"

break

else:

print "Todos os servidores atualizados com sucesso"

print "Finalizando o script"
