Para remover um registro, primeiro precisamos buscá-lo no banco de dados e depois fazer o delete.

Com essas duas linhas fizemos um delete em nosso banco de dados na tabela usuários, mas mesmo usando o SQLAlchemy, ainda é necessário executar um commit para efetivar a inserção.

usuario = session.query(Usuario).filter_by(id==1).first()

session.delete(usuario)

session.commit()
