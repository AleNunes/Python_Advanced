Dicionários

18

Dicionários são variáveis que armazenam dados do tipo Chave: Valor (formato idêntico ao JSON). São indispensáveis quando se trabalha com API's.

produto = {

“nome”:”Tenis”,

“valor”:59.30,

“categoria”:”social”,

“marca”:”sinvuca”,

}

print produto[“nome”]
