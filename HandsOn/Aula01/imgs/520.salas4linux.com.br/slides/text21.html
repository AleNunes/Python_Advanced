Operadores

Existem também os operadores lógicos. Eles são utilizados para testes condicionais:

  • var1 and var2: retorna True se var1 e var2 forem TRUE;
  • var1 or var2: retorna True se num1 ou num2 forem TRUE;
  • not var: retorna TRUE se var for False;
  • num1 == num2: retorna True se num1 e num2 forem iguais;
  • num1 != num2: True se num1 for diferente de num2 forem TRUE;
