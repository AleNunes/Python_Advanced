Banco de Dados

<número>

SQL é uma linguagem de programação declarativa, criada para gerenciar informações em um sistema gerenciador de banco de dados. É com a SQL que podemos criar, alterar, remover e resgatar informações de nosso banco de dados. Existem vários tipos de comandos separados por categoria. Os principais são:

DDL: Data Definition Language, CREATE, ALTER,DROP

DML: Data Manipulation Language, INSERT,UPDATE,DELETE

DQL: Data Query Language, SELECT,SHOW
