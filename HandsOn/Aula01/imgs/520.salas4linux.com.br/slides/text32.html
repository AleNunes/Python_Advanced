Laços de Repetição

12

No caso da linguagem python o laço de repetição for pode ser combinado com o comando else. Isso permite que você execute um comando se não encontrar um valor dentro do loop, por exemplo.

for item in lista:

if item == “python”:

print “Encontrado!”

break

else:

print “Não foi encontrado o python”
