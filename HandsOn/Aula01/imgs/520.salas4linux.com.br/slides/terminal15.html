Strings

  • Uma string é, nada mais nada menos, que um texto. Para declarar uma string, utilizamos aspas simples ( ' ) ou aspas duplas ( " ).
  • Para concatenar uma string com uma variável string utiliza-se o sinal de mais (+).
  • Para concatenar uma string com outros tipos utiliza-se a vírgula ( , ).

string = “O Nome do Criador do Python e: ”+nome

string = “A idade dele é: ”,idade

nome = “Guido”

idade = 59
