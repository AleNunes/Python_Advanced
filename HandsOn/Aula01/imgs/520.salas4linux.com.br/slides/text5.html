Quem usa Python?


Notas:

Python é bem popular no mundo da programação Web e ficou muito mais famoso com o surgimento do framework Django, concorrente direto do Rails (framework web para Ruby).

Django: https://www.djangoproject.com/

Outro framework que tem ganhado bastante espaço é o Flask, devido a sua simplicidade e o seu jeito “Pythonistico” de fazer as coisas, fazendo com que o seu projeto seja o mais modular possível através de Blueprints.

Flask: http://flask.pocoo.org/

Para aplicações Desktop temos os módulos PyGTK que cria interfaces para ambientes desktops baseados em Gnome:

PyGTK : http://www.pygtk.org/

E pyQt para ambientes baseados em ambiente KDE:

PyQT: https://wiki.python.org/moin/PyQt
