Operadores

Operadores relacionais são utilizados para comparações, retornando true ou false em seus resultados, são eles:

  • == = Igual
  • != ou <> = Não igual
  • < = Menor
  • <= = Menor ou igual
  • > = Maior
  • >= = Maior ou igual
  • is = Compara Booleano
