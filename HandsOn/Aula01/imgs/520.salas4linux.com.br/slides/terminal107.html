Para buscar um dado no banco, podemos utilizar o atributo query do objeto session,

Na primeira linha buscamos no banco de dados apenas um registro. Foi usado o método first() que retorna apenas o objeto Usuário. Caso ele não seja especificado, será retornada uma lista, mesmo que só exista um usuário com esse id.

Na segunda linha buscamos todos os registros na tabela Usuário.

usuario = session.query(Usuario).filter(Usuario.id==1).first()

usuarios = session.query(Usuario).all()
