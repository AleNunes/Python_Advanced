Criando Atalhos de Comandos

  • Entendendo Alias:

1# unalias -a

2# ls /

3# ls --color /

Servidor: Máquina Linux Interna / Webserver Interno

  • Criando Alias:

4# alias ls='ls --color'

5# ls /

  • Listando os Alias:

6# alias

Alias no Shell são como Apelidos de Comandos.

É a forma mais fácil de personalizar comandos do jeito que o administrador precisa.

Um alias pode conter apenas um comando com opções, ou ter um conjunto de comandos.


Notes:

Alias

Um recurso do “shell” que facilita muito a vida do administrador é a definição de “aliases” para comandos, os “aliases” permitem que um determinando comando seja invocado através de um apelido, seja para facilitar o acesso a sua execução ou para traduzir em formato simples um comando complexo.

Imagine que um determinado usuário gosta de utilizar o comando “ls” sempre com os parâmetros “--color” e “-la”. O que seria dele se toda vez que fosse executá-lo tivesse que escrever o comando com todos os parâmetros?!

NOTA:

Muitas distribuições já possuem uma série de alias pré-configurados.
