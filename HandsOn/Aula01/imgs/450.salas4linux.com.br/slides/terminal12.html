Mapeamento do Sistema

  • Nome do Servidor:

1# hostname

  • Endereço IP do Servidor:

2# ip address

3# ip a

4# ifconfig

  • Gateway Padrão:

5# ip route show

6# route -n

  • Servidores DNS:

7# cat /etc/resolv.conf

  • Serviços disponíveis:

8# ss -nltp

9# netstat -nltp

  • Versões dos serviços:

10# dpkg -l | grep apache

11# rpm -qa | grep httpd

Servidor: Webserver Interno / Linux Interna

Notes:

É realmente importante manter os pacotes atualizados?

É muito importante que as principais aplicações de um determinado servidor estejam sempre em sua última versão, evitando vulnerabilidades que comprometam a segurança do sistema.

Mantenha sempre as versões das aplicações documentadas e acompanhe no site do fornecedor dessa aplicação as atualizações de versões.
