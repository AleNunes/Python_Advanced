Comando lspci:

  • O primeiro item mostrado é o ID do dispositivo, então com o ID do dispositivo em mãos podemos saber mais detalhes sobre o mesmo, como o mapeamento da memoria usando o comando lspci -s ID -v;
  • Veja a saída do comando com o ID 00:03.0 (Ethernet controller):

1# lspci -s 00:03.0 -v
