Introdução a Redes

Servidor: Webserver Interno

Explorando o comando IP:

  • Configuração de IP:

1# ip addr add 192.168.200.20/24 dev eth0

2# ip a show eth0

  • Tente fazer um ping no IP da interface:

3# ping -c4 192.168.200.20

Notes:

Configurando IP e Máscara:

O comando “ifconfig” calcula automaticamente a máscara, mas se você precisar configurar uma máscara diferenciada, você deve usar o parâmetro “netmask” e a máscara correspondente.
