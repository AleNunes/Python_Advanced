Introdução a Redes

Um computador precisa de um roteador padrão (Gateway) para navegar em outras redes, ou seja, para sair para a internet. No Linux, o comando que define de forma temporária o gateway do sistema é o comando ip route.

Servidor: Webserver Interno

  • Deletando o Roteador Padrão:

1# ip route del default via 192.168.200.1

2# ip route show ; ping -c4 8.8.8.8

3# ip route add default via 192.168.200.1 ; ping -c4 8.8.8.8
