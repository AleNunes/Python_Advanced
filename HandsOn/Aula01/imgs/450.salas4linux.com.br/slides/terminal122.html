Introdução a Redes

Servidor: Webserver Interno

1# ip add a 10.10.0.20/16 dev eth0 label eth0:0

2# ip a show eth0

3# ping -c4 10.10.0.20

4# ip addr del 10.10.0.20/16 dev eth0:0

5# ip a show eth0

6# ping -c4 8.8.8.8

  • Configuração de IP Virtual: é uma prática usada quando existe uma limitação de placas físicas, porém uma necessidade de interligar redes com diferentes range de IPs. Você pode criar diversos alias para uma mesma placa.
