9# dpkg -S /usr/bin/top

Opções do dpkg

-l → (list) Lista todos os programas instalados;

-s → (status) Verifica o status de um programa instalado;

-S → (search) Busca o nome do programa que instalou determinado comando, arquivo ou biblioteca;


Notes:

DPKG (Debian Package) - Gerenciador de Pacotes do Debian

O DPKG é um programa que é a base do Sistema de Gerenciamento de Pacotes para distribuições GNU/Linux baseadas em Debian. Criado por Ian Jackson em 1993, o DPKG é usado para instalar, remover e fornecer informações sobre os pacotes .deb.
