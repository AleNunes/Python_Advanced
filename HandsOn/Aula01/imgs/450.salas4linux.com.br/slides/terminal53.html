Editores de Texto

Praticando o Vim:

Servidor: Webserver Interno

1# vim primeiro.txt

Divide a Tela do vim

:vsplit

Para Navegar entre as telas:

CTRL + W

Para Abrir um novo arquivo sem sair do vim:

:e nome_do_arquivo

Para Executar um comando sem sair do vim:

:r comando

Funcionalidades Básicas:

Dividir a Tela: split

Dividir Horizontalmente: vsplit

Executar um comando: e

Executar um comando: !

Executar e Inserir: r!

Outras formas de salvar e sair:

:x

Shift ZZ

Notes:

Funcionalidades do Vim

As funcionalidades do Vim é extremamente útil para a praticidade de um SysAdmin, tendo como objetivo a agilidade para a edição de arquivos.
