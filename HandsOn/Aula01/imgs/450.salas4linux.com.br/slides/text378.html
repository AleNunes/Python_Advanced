Gerenciamento Avançado de Partições

Servidor: Webserver Interno

  • Para visualizar as partições swaps ativas:

2# swapon -s

  • Dê um nome para a nova partição SWAP:

3# mkswap -L SWAP /dev/sdb3

1# swapoff /dev/sdb3

  • Para desabilitar a partição swap:

Montagem automática no boot


Notes:

O comando swapon tem a opção -p que habilita a prioridade:

-p, --priority <nº>

Quanto maior o número maior a prioridade que pode variar entre 0 e 32767.
