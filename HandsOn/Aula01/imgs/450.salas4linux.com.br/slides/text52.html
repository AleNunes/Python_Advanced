Editores de Texto

Praticando o Vim:

Servidor: Webserver Interno

1# cat primeiro.txt

2# vim primeiro.txt

< Com o Cursor na Frase tecle yy para copiar>

_GNU/Linux is Open Source

< Em seguida tecla p para colar a linha>

_GNU/Linux is Open Source

_GNU/Linux is Open Source

< Substitua OpenSource por Software Livre>

_GNU/Linux is Free

_GNU/Linux is Free

Funcionalidades Básicas:

Copiar Linhas: yy ou 10yy

Colar Linhas: p

Deletar Linhas: dd ou 10dd

Numerar Linhas: :set number

Substituição: :%s/Origem/Destino/g

Ir para Última Linha: Shift+G

Ir para Primeira Linha: gg

Desfazer Ações: u

Refazer Ações: CTRL+R

Buscar Palavra: /palavra

Buscar a próxima: n


Notes:

Funcionalidades do Vim
