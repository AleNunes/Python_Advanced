Introdução a Redes

Servidor: Webserver Interno

Explorando o comando IP:

  • Para verificar as interfaces com o comando IP:

1# ip address ou ip a

2# ip a show eth0

  • Desative a interface eth0:

3# ip link set eth0 down

4# ip addr del 192.168.200.20/24 dev eth0

5# ip a show eth0

  • Ative a interface eth0:

6# ip link set eth0 up

7# ip a show eth0
