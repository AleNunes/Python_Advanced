Resolução de Nomes

Servidor: Webserver Interno / Linux Interna

1# cat /etc/resolv.conf

2# ping -c4 gateway.dexter.com.br

3# vim /etc/hosts

....

192.168.200.1 gateway.dexter.com.br gateway

4# ping -c4 gateway

5# ping -c4 gateway.dexter.com.br

  • Existem basicamente 2 formas de resolver nomes no Linux:
    • Configurar um servidor DNS (/etc/resolv.conf);
    • Definir resolução manual (/etc/hosts).
