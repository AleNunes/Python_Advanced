Projeto Dexter Web no OpenShift da RedHat:

Sobre o Projeto Dexter Web:

O departamento de TI da Dexter Courier contratou uma nova plataforma Cloud chamada OpenShift para hospedar seu site que utiliza Apache, PHP e Banco de dados MySQL.

O projeto esta divido em 3 Etapas distribuída da seguinte maneira

Etapa I - Curso 4450: Criar a conta e instâncias no ambiente do OpenShift.

Etapa II - Curso 4451: Criar o banco de dados e tabela na instância do OpenShift.

Etapa III - Curso 4452: Configurar a página PHP e fazer upload do site para a instância do OpenShift.

As atividades desse projeto serão executadas pelo aluno extra classe.
