Definindo o Comportamento do Vim:

set ic: ignorar as buscas case sensitive;

set number: definir numeração de Linha;

syntax on: deixar o Vim colorido;

set hlsearch: destacar as palavras encontradas em uma busca.

set bg=dark: Deixa as fontes em negrito.

Notes:

Personalizando o Vim:

A grande maioria dos serviços em “Unix” são configurados através de arquivos de configuração, o “vim” não seria diferente. Seu arquivo de configuração é o “vimrc”.

Para configurar o seu editor de texto, basta adicionar as funcionalidades desejadas no arquivo de configuração, encontrado em /etc/vim/vimrc no Debian e /etc/vimrc em distribuições Red Hat.
