Introdução a Redes

Um computador precisa de um roteador padrão (Gateway) para navegar em outras redes, ou seja, para sair para a internet. No Linux, o comando que define de forma temporária o gateway do sistema é o comando route.

Servidor: Máquina Linux Interna

  • Adicionando o Roteador Padrão:

1# ping -c4 8.8.8.8 ; route -n

2# route add default gw 192.168.200.1

3# ping -c4 8.8.8.8 ; route -n

  • Deletando o Roteador Padrão:

4# route del default

5# ping -c4 8.8.8.8 ; route -n

6# route add default gw 192.168.200.1
