Pergunta LPI

Qual destes comandos permite que você declare a variável LD_LIBRARY_PATH como uma variável global, para que as bibliotecas compartilhadas que estão em /usr/local/lib sejam válidas?

a. set LD_LIBRARY_PATH = /usr/local/lib

b. export LD_PRELOAD = /usr/local/lib

c. export LD_LIBRARY_PATH = /usr/local/lib

d. env LD_LIBRARY_PATH = /usr/local/lib

Resposta: c

Notes:

Alternativa C: RESPOSTA CORRETA!

O comando utilizado para exportar uma variável tornando-a global é o comando export, na situação descrita a variável a ser exportada é a variável LD_LIBRARY_PATH logo a alternativa correta é:

export LD_LIBRARY_PATH=/usr/local/lib
