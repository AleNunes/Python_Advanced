Manipulando Hardware e Dispositivos

Servidor: Webserver Interno

Dispositivos de Armazenamento:

  • Foi adicionado ma máquina Webserver Interno um segundo disco /dev/sdb para realizarmos os particionamentos e criamos um diretório para backup, uma partição nova para dados e uma swap slave.
  • Execute os comandos abaixo para verificar o tamanho do disco:

1# cat /proc/partitions

2# dmesg | egrep sd[a-b]

3# fdisk -l | grep sdb
