Documentação e Python

Agora todas as vezes que você modificar os comentários do código, você deverá rodar o comando sphinx-apidoc de novo para atualizar.

Os parâmentros –full, -H e -A são somente na primeira vez.

Agora o comando fica assim:

sphinx-apidoc -o docs . -f

O -f continua para que ele possa sobrescrever os arquivos.
