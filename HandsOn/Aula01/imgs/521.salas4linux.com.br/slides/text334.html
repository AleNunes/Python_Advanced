Criando Dashboard com Flask

Criamos uma ferramenta que automatiza a criação de infraestrutura, criação de jobs e repositório no gitlab. No entanto, ainda temos que realizar um trabalho manual na criação de um arquivo json que serve como entrada de dados para a nossa ferramenta.

Isso pode ser facilitado através de um Dashboard que podemos criar utilizando Python, HTML e Javascript.
