Configurando Autenticação

É importante também gerenciar os security groups, uma vez que dependendo da aplicação que você for publicar em uma instância em EC2, pode-se precisar de portas diferentes das que vem na política padrão.

Para criar um security group utilize a seguinte instrução:

sg = ec2.create_security_group(

GroupName=”Python-SG”,

Description=”Grupo criado atraves do boto3”)

A variável SG irá armazenar o valor do security group. Agora você pode criar uma instância e já associá-la a esse security group.
