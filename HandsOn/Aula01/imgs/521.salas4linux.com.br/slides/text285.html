Automatizando Deploy

Essa classe irá precisar também de um método get que irá retornar o arquivo yaml já convertido.

Exemplo:

def get(self):

return self.book

A partir dos dados que essa classe retornar podemos fazer o Deploy da nossa aplicação. Esse Deploy ficará dinâmico, uma vez que será necessário somente mudar o arquivo de Deploy para que seja alterada a sequência de comandos.
