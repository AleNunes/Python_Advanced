Construindo DexterOps

Para deixar o nosso sistema mais dinâmico, vamos criar um arquivo de configuração chamado config.cfg e vamos armazená-lo na raiz do nosso projeto. Por enquanto, o conteúdo do mesmo deve ser esse:

[gitlab]

server = gitlab.dexter.com.br

token = y5BAhu9Tdmi84XjSY2yR

Esses serão os dados utilizados para conectar no gitlab e o token de autenticação.
