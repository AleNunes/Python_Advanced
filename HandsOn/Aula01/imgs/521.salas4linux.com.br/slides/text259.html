Configurando Autenticação

Caso queira listar todos os security groups existentes, pode-se utilizar as seguintes instruções:

for s in ec2.security_groups.all():

print s.group_name, s.group_id

Assim você pode saber todos os security groups existentes sem precisar logar no console da AWS.

Outra coisa importante também é adicionar e remover regras desse security group.
