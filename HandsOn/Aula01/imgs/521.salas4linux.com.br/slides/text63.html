Criando Rotas e Views

Agora que já temos as nossas views, precisamos fazer com que elas recebam dados no formato json e enviem dados no mesmo formato.

Vimos que o python tem muita facilidade em trabalhar com esse formato, uma vez que temos os dicionários, que são praticamente iguais.

O flask possui um módulo que faz a conversão de um dicionário para o formato JSON e envia o response para o usuário.
