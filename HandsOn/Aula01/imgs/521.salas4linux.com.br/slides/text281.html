Automatizando Deploy

Vamos chamar o nosso arquivo de Dexterbook.yml.

O conteúdo desse arquivo pode ficar da seguinte maneira:

application: Terminus

repo: https://github.com/AlissonMMenezes/Terminus.git

deploy-sequence:

commands:

- apt-get update

- apt-get -f install python -y

- apt-get -f install python-dev -y

Siga a sequência de IF's sendo um para cada comando de deploy.
