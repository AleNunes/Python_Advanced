A primeira ação para inserir um registro é criar uma instância da classe que já realizada para gerar a tabela no banco de dados.

Com essas duas linhas fizemos um insert em nosso banco de dados na tabela usuários, mas mesmo usando o SQLAlchemy, ainda é necessário executar um commit para efetivar a inserção.

usuario = Usuarios(id=1,nome='Alan Kay')

session.add(usuario)

session.commit()
