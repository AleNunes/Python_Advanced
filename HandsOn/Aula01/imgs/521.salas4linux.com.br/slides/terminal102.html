Agora que já temos definida a estrutura de uma tabela, já podemos criar o nosso banco de dados e efetuar as operações CRUD.

O comando Base.metadata.create_all() recebe como parâmetro uma conexão com banco de dados e cria a estrutura independente do banco que você estiver usando.
