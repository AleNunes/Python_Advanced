Automatizando Deploy

Temos agora um problema a resolver:

De onde podemos ler os comandos para serem executados durante o Deploy?

Uma opção seria utilizar um arquivo no formato YAML e podemos ter uma configuração parecida com o Ansible.

Para utilizar o YAML precisamos instalar um módulo chamado pyyaml que será responsável por fazer o “parser” do arquivo. Esse módulo irá converter o YAML em um dicionário e podemos trabalhar em um formato parecido com o json.

Para instalar o módulo utilize o seguinte comando:

# pip install pyyaml
