Jenkins e Python

1

O primeiro passo é criar uma conexão com o servidor jenkins.

Abaixo um exemplo de código que faz essa conexão:

Import jenkins

server = jenkins.Jenkins(“http://jenkins.dexter.com.br:8080”)

print server.get_version()

2

Salve o arquivo como devops.py e o execute.

Se tudo deu certo, aparecerá a versão do jenkins na tela.
