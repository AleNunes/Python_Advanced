Integrando Docker e Python

Uma vez que o container não é mais necessário, pode-se exclui-lo também através de API. No entanto, é necessário parar um container caso ele esteja sendo executado para só depois efetuar a sua exclusão.

10

Para parar um container:

cli.stop(container.get(“Id”))

11

Para remover o container:

cli.remove_container(container.get(“Id”))
