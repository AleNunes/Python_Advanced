Network com Docker

1

Crie uma rede interna do Docker com o nome de Dexterlan.

1# docker network create dexterlan

2

Crie o container de teste utilizando a nova rede criada.

(Execute na mesma linha!)

2# docker run -it --name node01 --hostname node01 --net dexterlan net-teste /bin/bash

3

É possível também definir um IP fixo para o container que será executado.

3# docker run -it --name node01 --hostname node01 --net dexterlan --ip 172.18.0.2 net-teste /bin/bash
