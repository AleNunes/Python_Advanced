Automatizando Deploy

A primeira coisa que vamos fazer é automatizar a criação de containers no Docker.

Na pasta da aula 08, crie um diretório chamado Module. Nele vamos colocar os módulos específicos para realizar a integração.

Dentro desse diretório crie um arquivo chamado DockerModule.py. Nele faça o import da classe Client do módulo Docker.

# from docker import Client

cli = Client(base_url=”tcp://192.168.0.2:2376”)

Nessa segunda linha efetuamos a conexão com o servidor do Docker. Agora podemos criar um container.
