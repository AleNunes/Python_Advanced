Consumindo uma REST API pelo Navegador

Vamos agora efetuar alguns testes para ver na prática como as requisições de API's funcionam.

No Firefox existe uma extensão chamada RestClient, que nos permite enviar requisições dos tipos GET, POST, PUT e DELETE para uma determinada URL e setar o cabeçalho da aplicação que, no nosso caso, será um application/json. Este é o formato de dados que vamos trabalhar.
