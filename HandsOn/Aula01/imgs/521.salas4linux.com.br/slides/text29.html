Consumindo uma REST API com Python

Se eu quiser buscar por um usuário específico, tenho que seguir a mesma lógica utilizada no RestClient:

print requests.get(“http://127.0.0.1/usuarios/1/”)

O retorno de ambos os comandos, será <Response [200] >, pois ele retorna somente o Status da requisição. No caso 200, esta requisição foi realizada com sucesso.
