Integrando Docker e Python

1

Para integrar o Docker com a linguagem de programação Python, é necessário instalar um módulo chamado docker-py. Para fazer essa instalação execute o seguinte comando:

pip install docker-py

2

Agora, no diretorio da Aula05, crie um arquivo chamado:

integrando_docker.py

Neste diretório vamos ver as operações que podem ser realizadas com o Docker através da API.
