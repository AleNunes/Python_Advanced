Jenkins e Python

Para adicionar um novo filho à um elemento já existente, precisa-se primeiro criar o elemento e depois fazer um append ao elemento PAI. O ElementTree, que é a função que importamos do módulo lxml, considera o arquivo xml com listas, então, pode-se usar as mesmas operações das listas para tratar esse tipo de arquivo.

Exemplo:

novo_comando = etree.Element(“command”)

novo_comando.text = ”echo testando novo comando”

builder.append(novo_comando)
