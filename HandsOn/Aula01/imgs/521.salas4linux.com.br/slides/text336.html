Criando Dashboard com Flask

Para renderizar um html, é necessário criar uma pasta chamada templates no mesmo diretório onde criamos o Dashboard.py.

Agora teremos que alterar duas linhas:

from flask import Flask,render_template

O return ficará assim:

return render_template(“index.html”)

Esse index.html deve ser criado dentro da pasta templates e pode ter qualquer conteúdo.
