Entendendo a REST API

JSON significa JavaScript Object Notation. É um protocolo leve para transferência de dados e o seu formato tem o mesmo nome (JSON).

Esse formato foi criado por Douglas Crockford, muito conhecido pela contribuição de ferramentas voltadas para JavaScript como JSLint e JSMin.

Hoje esse formato JSON é o mais utilizado na

criação de REST API's pelo seu formato limpo e

facilidade na integração com AJAX.
