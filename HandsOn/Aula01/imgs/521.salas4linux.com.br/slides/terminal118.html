Entendendo o MongoDB

<número>

MongoDB é um banco de dados noSQL de código aberto, que armazena os seus dados utilizando o formato JSON, com o objetivo de ter alta performance.

Os banco de dados no SQL geralmente ocupam mais espaço em disco por não terem o recurso de JOIN entre os dados, porém, a leitura e a escrita deles são bem superiores.

apt-get install mongodb

service mongodb start

mongo

O último comando serve para acessar o console do Mongodb.
