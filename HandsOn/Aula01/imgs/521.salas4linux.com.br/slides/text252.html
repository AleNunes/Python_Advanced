Configurando Autenticação

Agora você já pode iniciar a criação do seu script. Para começar é necessário importar o módulo Boto3.

# import boto3

Esse mesmo módulo serve para integrar qualquer um dos serviços da Amazon. No nosso caso, vamos utilizar o EC2.

ec2 = boto3.resource(“ec2”)

Agora você pode utilizar o objeto EC2 para realizar as operações na Amazon.
