Configurando Autenticação

Para criar uma nova instância, pode-se utilizar a seguinte instrução:

new_instance = ec2.create_instances(

ImageId='ami-b2e3c6d8',

MinCount=1,

MaxCount=1.

SecurityGroupIds=[“sg-dc918cb6”])

Para validar se o security group foi adicionado, você pode utilizar a seguinte instrução:

Instance = ec2.instances.filter(InstanceId(InstanceIds=[“i-c7b53844”]))

print next([ i.security_groups for i in instance ])
