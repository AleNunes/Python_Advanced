Python e Gitlab

Agora podemos utilizar dos recursos do list comprehension para buscar o id do projeto e dos usuários que queremos adicionar aos repositórios.

projeto = [ p for p in getprojects() if p['name'] == 'projeto-integracao']

membro = [m for m in getusers() if m['username'] == 'desenvolvedor1']

addprojectmember(projeto[0]['id'],membro[0]['id'])

Na função addprojectmember, o access_level foi ajustado como 30 que é o valor equivalente a permissão de desenvolvedor do gitlab.
