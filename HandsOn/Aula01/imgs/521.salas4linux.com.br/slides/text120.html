Python e Gitlab

Para instalá-lo execute o seguinte comando:

# sudo pip install requests

Para fazer a integração do python com o gitlab, vamos utilizar um módulo conhecido chamado requests.

Feito isso, já podemos começar a construir as nossas integrações.

Então, na pasta da aula04 vamos criar um arquivo chamado gitlab_integration.py e vamos testar as funções mais utilizadas do gitlab.
