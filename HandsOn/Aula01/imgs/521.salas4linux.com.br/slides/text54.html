Criando Rotas e Views

Então, se quiser criar uma nova view, pode-se simplesmente copiar a função e a rota e colocar logo abaixo.

Exemplo:

@app.route(“/usuarios/”)

def index_usuarios():

return “Aqui serao gerenciados os usuarios”

Agora é só salvar o seu arquivo e tentar acessar pelo navegador com a mesma URL, porém, colocando-se /usuarios/ para ver se aparecerá essa mensagem na tela.
