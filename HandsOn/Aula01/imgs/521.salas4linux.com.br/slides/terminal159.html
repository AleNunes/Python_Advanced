O Projeto Dexter é um sistema utilizado para fazer a entrega de serviços dentro de containers em docker. Nele podemos definir quais imagens do docker serão usadas e associar essas imagens aos produtos que vamos fornecer. Como neste módulo não vimos o conceito de API, todos os comandos deverão ser executados remotamente através do SSH.
