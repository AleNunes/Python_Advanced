Flask - MongoEngine

MongoEngine é uma ORM (Object Relationship Mapping) para MongoDB.

Como temos uma API simples e enviamos e recebemos dados pelo formato JSON e o MongoDB armazena esses dados no mesmo formato, podemos aproveitar e unir as duas coisas.

No primeiro curso de Python foi mostrado o uso do SQLAlchemy, que é um ORM, porém voltado para banco de dados relacionais.
