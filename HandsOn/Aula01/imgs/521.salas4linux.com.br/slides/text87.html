Flask - MongoEngine

No campo e-mail foi colocado um parâmetro unique=True. Esse parâmetro serve para dizer ao MongoDB que esse campo nunca poderá ser repetido, uma vez que dois usuários não podem utilizar o mesmo e-mail.

No caso do data_cadastro, foi passado como parâmetro: default=datetime.datetime.now().

Caso não seja passada a data de cadastro, ele automáticamente irá assumir que a data é a data e hora atual.

Não esqueça de importar o módulo datetime!
