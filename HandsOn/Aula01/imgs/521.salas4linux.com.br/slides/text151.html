Liberando API do Docker

1

Para liberar a API do Docker, você precisa editar o arquivo:

/etc/default/docker

2

Na linha em que está DOCKER_OPTS, deve-se adicionar a seguinte configuração:

-H tcp://0.0.0.0:2376

Com essa opção a API Docker pode ser acessada por qualquer endereço. Também é possível restringi-la para somente a rede interna ou somente para um IP em específico.
