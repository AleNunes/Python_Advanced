Formato dos Logs

No caso da aplicação o Log poderia ficar a seguinte maneira:

[ 30/03/2016 17:14 ] ERROR – Falhou ao criar usuário no gitlab

Podemos criá-lo seguindo exatamente esse formato:

[ DATA HORA ] NIVEL DO LOG – EVENTO OCORRIDO

Em arquivos de Log é muito comum também se guardar qual foi o usuário que executou determinado evento, mas para o nosso caso isso já será o suficiente, uma vez que somente nós vamos executar esse script.
