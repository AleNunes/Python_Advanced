Consumindo uma REST API com Python

Para fazer a atualização de um usuário cadastrado, segue-se exatamente a mesma lógica do post. Deverá ser alterado somente o método da request e especificar o ID do usuário no final da URL.

data = json.dumps({“nome”:”jubileu”,”email”:”jubileu@4linux.com.br”})

Headers = {'content-type','application/json'}

requests.put('http://127.0.0.1:5000/usuarios/1/',data=data,headers=headers)
