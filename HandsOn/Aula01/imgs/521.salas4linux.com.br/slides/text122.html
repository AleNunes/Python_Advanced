Python e Gitlab

Feito isso, você deverá fazer todas as requests no seguinte endereço:

http://gitlab.dexter.com.br/api/v3/RECURSO?private_token=SEUTOKEN

Onde está recurso, será o recurso da API que você vai utilizar.

Onde está token será o token que você pegou no gitlab.
