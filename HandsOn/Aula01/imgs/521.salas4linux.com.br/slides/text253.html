Configurando Autenticação

Para buscar todas as instâncias, pode-se utilizar o seguinte recurso:

instances = ec2.instances.all()

Essa instrução irá retornar uma lista. Para saber quais são os atributos que você pode listar, utilize o seguinte comando:

for i in instances:

print dir(i)

Isso irá mostrar todos os atributos dos objetos. Caso você já saiba o que quer ver, como por exemplo, o ID e o tipo da instância, pode-se utilizar as seguintes instruções:

print i.id, i.instance_type
