Jenkins e Python

Todas as alterações no XML foram feitas somente em tempo de execução do script. Assim que ele for finalizado essas alterações serão perdidas, então, é necessário fazer a persistência em um arquivo.

Podemos usar a mesma função open para escrever em um arquivo, mas ela recebe como parâmetro uma string. Então, é necessário converter o xml em string para depois gravar no arquivo.

Exemplo:

with open('job_modificada.xml','w') as f:

f.write(etree.tostring(root))
