Criando Dashboard com Flask

Agora se você acessar no navegador localhost:5000/provisionar/, será retornada a mensagem que informamos no return jsonify() no Dashboard.py.

Para receber os dados que foram enviados por ajax, volte na função provisionar e adicione o código:

print request.json()

O retorno virá em string, então, é necessário importar o módulo json e fazer o json.loads(request.json()) para que o python considere um dicionário.
