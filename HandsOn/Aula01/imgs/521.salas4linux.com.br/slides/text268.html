Integração Contínua

Este é o processo de Integração Contínua. Nele o desenvolvedor programa a aplicação, faz um commit para um servidor de controle de versão, depois um servidor de automação.

Ele também resolve os procedimentos necessários para compilar a Aplicação e o mesmo servidor faz os testes para garantir que erros não entrem em produção.
