Documentação e Python

Existe um site muito popular chamado Read The Docs, que hospeda a maioria das documentações de projetos em python. Caso você queira conhecer, acesse este link:

https://readthedocs.org/

O interessante é que ele disponibiliza WebHook para você integrar ao seu repositório. Caso você faça um novo git push, o readthe docs é acionado automaticamente e faz o deploy da sua documentação.
