Python e Gitlab

Ao criar uma integração com o Gitlab, a primeira coisa a se fazer é ver o token para conexão com o servidor.

Logue com o seu usuário e clique em Profile Settings > Account

E você verá o token de autenticação.

Copie e esse token e coloque no seu código:

import requests

token = “yBAhu9Tdmi84XjSY2yR”
