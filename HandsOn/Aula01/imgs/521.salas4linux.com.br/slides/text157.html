Integrando Docker e Python

6

Para criar um novo container, pode-se usar a seguinte função:

cli.create_container(image=”webservercloud”,command=”/bin/bash”)

A função create_container() recebe todos os parâmetros que são passados via linha de comando. Por exemplo,se for necessário dar um nome e um hostname para a máquina, você pode passar os seguintes parâmetros:

name=”nome”,hostname=”exemplo.com.br”
