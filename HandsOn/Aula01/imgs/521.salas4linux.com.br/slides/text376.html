Arquivo de Log

Por padrão as distribuições Linux guardam os arquivos de Log dentro do diretório:

/var/log/

Esse é um padrão estabelecido pela FHS ( Filesystem Hierarchy Standard ), que tem como objetivo padronizar a estrutura de diretórios do Linux. No nosso caso vamos criar uma pasta chamada Logs e nela vamos armazenar essas informações.
