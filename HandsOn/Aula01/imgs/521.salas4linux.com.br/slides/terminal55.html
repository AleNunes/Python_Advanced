Módulo Paramiko

<número>

O SSH precisa que seja aceito o fingerprint quando acessamos um servidor pela primeira vez. Este fingerprint serve para garantir a autenticidade de um servidor.

Para carregar os fingerprints dos servidores já acessados, utilizamos o seguinte comando:

client.load_system_host_keys()

client.set_missing_host_key_policy(paramiko.AutoAddPolicy())

Com a segunda linha, o paramiko aceita o fingerprint automaticamente, caso seja a primeira vez que acessa um servidor.
