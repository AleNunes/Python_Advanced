Documentação e Python

É possível também alterar o tema da sua documentação. Um dos mais populares é o tema do Read The Docs.

Para usá-lo instale-o com o seguinte comando:

sudo pip install sphinx_rtd_theme

Agora no conf.py, dentro da pasta docs, faça o import:

import sphinx_rtd_theme

Procure pela linha que possui a variável html_theme e altere o valor para:

html_theme = “sphinx_rtd_theme”
