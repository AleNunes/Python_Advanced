Repositório Remoto com Git

Agora pegue a URL do seu repositório e faça um clone para a sua máquina.

Exemplo:

# git clone http://gitlab.dexter.com.br/devops/applicacao-exemplo.git

Foi criada uma pasta no diretório corrente com o mesmo nome do repositório. Você pode entrar nela agora e criar os seus arquivos.
