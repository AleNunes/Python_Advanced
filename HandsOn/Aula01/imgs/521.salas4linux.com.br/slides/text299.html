Construindo DexterOps

DexterOps é um sistema focado em realizar integrações. Nele podemos criar repositórios no gitlab, criar jobs no jenkins, criar containers no docker, fazer o deploy de aplicações, etc.

Já temos o código do DeployTools que foi construído na aula passada. Ele será reaproveitado nesta aula, assim, teremos um sistema completo de automatização e integração.
