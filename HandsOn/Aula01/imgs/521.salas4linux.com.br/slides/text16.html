Consumindo uma REST API pelo Navegador

Na opção URL, vamos colocar a URL para fazer as requisições que, no nosso caso, será 127.0.0.1:5000/usuarios/ e as variações.

Na opção Method, vamos testar os 4 principais: POST,GET,PUT e DELETE.

Em headers ajustaremos para: Content-Type:application/json.

Em Body estarão os dados que vamos enviar para o servidor.

Em Response teremos o retorno dos dados do servidor, assim saberemos se um comando foi executado com sucesso ou não.
