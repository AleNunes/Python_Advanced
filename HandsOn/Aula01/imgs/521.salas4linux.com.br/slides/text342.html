Criando Dashboard com Flask

É uma boa prática colocar todos os arquivos CSS e Javascript no final da página para melhorar a velocidade de conexão, uma vez que os elementos HTML serão carregados antes para só ao final aplicar os estilos CSS e os scripts em JavaScript.

No final da página, acima de </body>, faça o include do bootstrap.

<link rel=”stylesheet” href=”/static/css/bootstrap.css” />
