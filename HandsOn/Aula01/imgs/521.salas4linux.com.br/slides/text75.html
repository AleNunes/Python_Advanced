Entendendo o Model

O Model de uma aplicação é o equivalente ao modelo do seu banco de dados, que pode ser tanto em SQL como em noSQL.

O Banco de Dados serve para fazer a persistência dos seus dados para que possam ser utilizados posteriormente.

Como a nossa API a princípio é bem simples, só precisamos controlar usuários e grupos.

Precisamos, então, de duas entidades:

1

Usuários

2

Grupos
