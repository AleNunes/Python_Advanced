Jenkins e Python

3

Agora que a conexão foi estabelecida podemos criar uma Job.

Abaixo do server.get_version(), adicione o seguinte código:

print server.create_job(“Python-Job”,jenkins.EMPTY_CONFIG_XML)

4

Execute o script e veja que foi criada uma Job no Jenkins exatamente com o nome Python-Job.

O segundo parâmetro do método create_job(), deve receber um arquivo xml com as configurações da Job. No nosso caso, passamos o jenkins.EMPTY_CONFIG_XML para que seja criada uma Job totalmente em branco.
