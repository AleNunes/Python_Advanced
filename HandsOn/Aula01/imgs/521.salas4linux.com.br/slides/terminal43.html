Criando Funções

<número>

Uma função utiliza variáveis de seu escopo local, mas podemos acessar uma variável global utilizando a palavra reservada global.

def nomeFuncao(parametros):

global valor

echo valor

Atenção: Toda variável criada dentro de uma função possui um escopo local. Podemos acessar variáveis que foram criadas fora da função, ou seja, que possuem escopo global.
