Os mesmos métodos existem tanto para entrada padrão, saída padrão como para saída de erro.

stderr.channel.recv_exit_status()

Este método irá retornar um valor inteiro. Se o comando foi executado com sucesso, retorno 0, caso contrário, um valor maior.

stdout.read()

Esse método irá exibir a mensagem retornada pelo comando executado.
