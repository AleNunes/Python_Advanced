Integrando Docker e Python

3

A primeira coisa que precisamos fazer é conectar com o servidor do Docker. Para isso, execute as seguintes linhas de código:

from docker import Client

cli = Client(base_url=”tcp://192.168.0.2:2376”)

Lembre-se de colocar os códigos dentro de blocos try/except para poder tratar as mensagens caso ocorra algum erro.
