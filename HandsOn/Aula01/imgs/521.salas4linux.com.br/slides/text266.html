Integração Contínua

Uma vez que ela foi desenvolvida para qualquer uma dessas funções, pode-se fazer rotinas de integração contínua, integração entre sistemas, scripts de execução de tarefas e testes de aplicações.

A Linguagem Python é ótima para realizar este tipo de processo. Não só para publicar aplicações escritas em Python, como também pode ser utilizada para auxiliar no “Deploy” de qualquer tipo de aplicação.
