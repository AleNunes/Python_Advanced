Consumindo uma REST API com Python

A Linguagem Python possui módulos muito bons para consumir API's. Vamos utilizar um módulo chamado requests.

Para instalar o módulo execute o seguinte comando:

Com ele pode-se chamar as URL's para acessar os recursos, definir os métodos, definir o Content-Type e uma série de outros elementos.

# pip install requests
