Jenkins e Python

A parte repositório é relativamente simples de se resolver. É possível fazer um replace do valor que está atualmente no xml para o valor que você quer.

1

Vamos substituir o repositório pelo que foi criado na aula de Gitlab.

with open('xml_exemplo.py','r') as f:

config_xml = f.read().replace('git@gitlab.dexter.com.br',”git@gitlab.dexter.com.br:devops/aplicacao-exemplo.git”)

2

Feito isso você pode gravar o arquivo modificado em um arquivo temporário.

with open(“/tmp/temp_job.xml”,'w') as f: f.write(config_xml)
