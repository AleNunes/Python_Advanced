Criando Dashboard com Flask

Quando se cria aplicações web é muito comum usar javascript para fazer o controle dos campos, enviar requisições por ajax, entre outras rotinas.

Para facilitar o trabalho com javascript existe uma biblioteca chamada Jquery, que vamos baixar e utilizar em nosso projeto.

https://jquery.com/download/

Baixe a versão 1.12.2 e descompacte dentro de /static/js/.
