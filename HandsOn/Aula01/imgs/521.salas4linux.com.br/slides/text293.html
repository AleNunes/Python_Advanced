Automatizando Deploy

Agora crie um método chamado exec_command na classe DockerModule que deve receber dois parâmetros. Um deles é o nome do container e o outro é o comando que deverá ser executado dentro dele.

Para executar os comandos via ssh faça com que a classe DockerModule herde as propriedades da classe SSHModule.

No método exec_command do Docker o código deve ser este:

def exec_command(self,container,cmd):

print “[+] Runing command: %s”%cmd

command = “docker exec %s %s”%(container,cmd)

print self.execCommand(command)
